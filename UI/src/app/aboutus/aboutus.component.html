CovaX Systems

"A web application which supports health administrations in finding suitable locations in Frankfurt am Main, to establish the vaccination stations. The application provides recommendations based on the population and area of the different Ortsbezirke in Frankfurt and availability of healthcare institutions such as hospitals and clinics. The users are able to save their chosen locations so that they are able to progress without rework. The user interface of the system is carefully designed to provide seamless experience for the users."
